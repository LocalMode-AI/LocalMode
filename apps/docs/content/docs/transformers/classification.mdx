## Recommended Models

| Model | Size | Labels | Use Case |
|-------|------|--------|----------|
| `Xenova/distilbert-base-uncased-finetuned-sst-2-english` | ~67MB | POSITIVE/NEGATIVE | Sentiment analysis |
| `Xenova/twitter-roberta-base-sentiment-latest` | ~125MB | Positive/Neutral/Negative | Social media sentiment |
| `Xenova/toxic-bert` | ~110MB | Toxic/Non-toxic | Content moderation |

## Best Practices

<Callout type="tip" title="Classification Tips">
  1. **Choose task-specific models** — Sentiment models work best for sentiment, not topic detection
  2. **Check allScores** — Low confidence scores may indicate ambiguous input
  3. **Batch when possible** — `classifyMany()` is more efficient than multiple `classify()` calls
  4. **Cache the model** — Create the model instance once and reuse it
</Callout>
