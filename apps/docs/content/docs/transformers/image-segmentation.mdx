## Background Removal Example

Based on the [Background Remover](https://localmode.ai/background-remover) showcase app:

```typescript
import { transformers } from '@localmode/transformers';
import { segmentImage } from '@localmode/core';

const model = transformers.segmenter('briaai/RMBG-1.4');

async function removeBackground(imageDataUrl: string) {
  const { masks } = await segmentImage({
    model,
    image: imageDataUrl,
    abortSignal: controller.signal,
  });

  // Find the main subject mask (e.g., 'person', 'wall', etc.)
  // Apply the mask to create a transparent background
  return applyMaskToImage(imageDataUrl, masks);
}
```

## Best Practices

<Callout type="tip" title="Segmentation Tips">
  1. **License** — RMBG-1.4 is non-commercial only; commercial use requires a paid license from BRIA AI. Use `segformer-b0` (MIT) for commercial projects.
  2. **Purpose-built for background removal** — RMBG-1.4 excels at foreground/background separation
  3. **Multiple masks** — Each segment in the image gets its own mask and label
  4. **Combine masks** — Merge related masks for background removal (e.g., combine all non-subject masks)
  5. **Canvas processing** — Use Canvas API to apply masks for transparency effects
</Callout>
