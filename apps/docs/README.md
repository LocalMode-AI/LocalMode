# LocalMode Documentation

A Next.js documentation site built with [Fumadocs](https://fumadocs.dev) for the LocalMode packages.

## Overview

This app serves as the documentation hub for the LocalMode ecosystem - a collection of packages enabling browser-based AI/ML capabilities without server dependencies.

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the docs.

## Structure

- `content/docs/` — MDX documentation files
- `src/app/docs/` — Documentation pages and layout
- `src/app/(home)/` — Landing page
